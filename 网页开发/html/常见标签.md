## 段落标签
### 特点
- 块级元素，前后自动换行
- 用于包裹文本段落
- 默认有上下外边距

## 语法
``` html title:段落标签
<p>这是一个段落文本</p>
<p align="center">居中段落（HTML4写法，建议用CSS替代）</p>
```

### 常用属性
- 全局属性（class/id/style等）
- align（已过时，建议用CSS text-align替代）

### 默认样式
``` css 
p {
  display: block;
  margin-top: 1em;
  margin-bottom: 1em;
}
```

## 链接标签
### 特点
- 行内元素，可嵌套其他元素
- 用于页面跳转/锚点定位
- 必须用 href 指定目标地址

## 语法
``` html title:链接标签
<a href="https://example.com">外部链接</a>
<a href="#section2">跳转到第二节</a>
<a href="mailto:contact@example.com">发送邮件</a>
```

### 常用属性
- `href`：指定链接地址
- `target`：_blank（新窗口）/ _self（当前窗口）
- `download`：强制下载文件

### 默认样式
- 未访问：蓝色文字+下划线
- 已访问：紫色文字
- 鼠标悬停：手型光标+文字变色

## 图片标签
### 特点
- 自闭合行内元素
- 必须包含 src 和 alt 属性
- 支持 PNG/JPG/GIF/SVG 等格式

## 语法
``` html title:图片标签
<img src="image.jpg" alt="图片描述">
<img src="logo.png" width="200" height="100">
```

### 常用属性
- `src`：图片路径（必需）
- `alt`：替代文本（必需）
- `width`/`height`：显示尺寸
- `title`：悬停提示文本

### 默认样式
- 无边框（除非图片加载失败）
- 根据原始尺寸显示

## 列表标签
### 特点
- 块级容器元素
- 包含有序列表（ol）和无序列表（ul）
- 列表项使用 `<li>` 标签

## 语法
``` html title:列表标签
<ul>
  <li>项目一</li>
  <li>项目二</li>
</ul>

<ol start="5">
  <li>第五项</li>
  <li>第六项</li>
</ol>
```

### 常用属性
- `type`（ul）：disc/circle/square
- `type`（ol）：1/A/a/I/i
- `start`（ol）：起始编号

### 默认样式
- ul：左内边距40px + 圆点标记
- ol：左内边距40px + 数字编号
- li：display: list-item

## 容器标签
### 特点
- `<div>` 块级通用容器
- `<span>` 行内文本容器
- 常用于CSS布局和内容分组

## 语法
``` html title:容器标签
<div class="container">
  <span>高亮文本</span>
</div>
```

### 常用属性
- 全局属性
- 无特殊属性（主要通过CSS操作）

### 默认样式
``` css
div {
  display: block;
}
span {
  display: inline;
}
```

## 表格标签
### 特点
- 结构化数据展示
- 由 `<table>` `<tr>` `<td>` 组成
- 支持表头 `<th>` 和标题 `<caption>`

## 语法
``` html title:表格标签
<table border="1">
  <tr>
    <th>姓名</th>
    <th>年龄</th>
  </tr>
  <tr>
    <td>张三</td>
    <td>25</td>
  </tr>
</table>
```

### 常用属性
- `border`：表格边框
- `cellpadding`：单元格内边距
- `cellspacing`：单元格间距

### 默认样式
- 无边框（除非指定border）
- 表格宽度自适应内容

## 表单标签
### 特点
- 用户输入数据收集
- 包含 `<form>` `<input>` `<textarea>` 等
- 需配合后端处理数据

## 语法
``` html title:表单标签
<form action="/submit" method="POST">
  <input type="text" name="username">
  <input type="submit" value="提交">
</form>
```

### 常用属性
- `action`：表单提交地址
- `method`：GET/POST
- `enctype`：数据编码方式

### 默认样式
- 块级元素
- 有默认外边距

---

## 按钮标签
### 特点
- 可点击的交互元素
- 三种类型：submit/reset/button
- 行内块级显示

## 语法
``` html title:按钮标签
<button type="submit">提交</button>
<button type="button" onclick="alert()">点击</button>
```

### 常用属性
- `type`：按钮类型
- `disabled`：禁用状态
- `autofocus`：自动聚焦

### 默认样式
``` css
button {
  display: inline-block;
  padding: 6px 12px;
}
```

## 输入框标签
### 特点
- 多种输入类型（text/password/email等）
- 自闭合行内元素
- 支持数据验证

## 语法
``` html title:输入框标签
<input type="text" placeholder="请输入">
<input type="password" required>
<input type="checkbox" id="agree">
```

### 常用属性
- `type`：输入类型（默认text）
- `name`：提交字段名
- `value`：默认值
- `placeholder`：提示文本

### 默认样式
- 边框：1px实线
- 高度：根据字体大小调整
