## vertical-align(垂直对齐)
### 介绍
简单说他的作用就是让**行内块和行内元素对齐**或者消除图片**去除图片底侧空白缝隙**
### 用法
| 参数值                | 描述                      | 适用元素       |
| ------------------ | ----------------------- | ---------- |
| `baseline`         | 默认值。元素基线与父元素基线对齐        | 行内元素、表格单元格 |
| `sub`              | 将元素垂直对齐为下标              | 行内元素       |
| `super`            | 将元素垂直对齐为上标              | 行内元素       |
| `top`              | 元素顶部与行中最高元素的顶部对齐        | 行内元素、表格单元格 |
| `text-top`         | 元素顶部与父元素字体顶部对齐          | 行内元素       |
| `middle`           | 元素中部与父元素中部对齐            | 行内元素、表格单元格 |
| `bottom`           | 元素底部与行中最低元素的底部对齐        | 行内元素、表格单元格 |
| `text-bottom`      | 元素底部与父元素字体底部对齐          | 行内元素       |
| `length` (如 `2px`) | 将元素相对于基线升高或降低指定长度       | 行内元素、表格单元格 |
| `%` (如 `20%`)      | 相对于行高(line-height)的百分比值 | 行内元素、表格单元格 |
| `inherit`          | 继承父元素的 vertical-align 值 | 所有元素       |
###  特殊说明
1. **表格单元格**：`vertical-align` 的 `top`、`middle` 和 `bottom` 分别表示单元格内容的顶部、中部和底部对齐
2. **行内元素**：影响元素相对于基线的垂直位置
3. **负值**：允许使用负值，可以将元素降低到基线以下
4. **块级元素**：不适用，除非设置为 `inline-block` 或 `table-cell`
## 过渡属性(transition)
### 特点
- 过渡的属性可以是具体的 CSS 属性
- 也可以为 all（两个状态属性值不同的所有属性，都产生过渡效果）
- transition 设置给元素本身
### 语法
``` css title:过渡属性的
div {
  width: 200px;
  height: 200px;
  transition: all 1s;
  background-color: red;
}

div:hover {
  width: 500px;
  height: 500px;
}
```

## 透明度(opacity)
作用：设置**整个元素的透明度**（包含背景和内容）
属性名：opacity
属性值：0 – 1
- 0：完全透明（元素不可见）
- 1：不透明
- 0-1之间小数：半透明
## 光标类型(cursor)

作用：鼠标悬停在元素上时指针显示样式
属性名：cursor
属性值:

| 属性值            | 描述                          | 示例外观              |
|-------------------|-------------------------------|-----------------------|
| `auto`           | 浏览器自动决定光标样式         | (根据上下文变化)      |
| `default`        | 默认光标（通常是箭头）         | ➚                   |
| `pointer`        | 手形指针（表示链接可点击）     | 👆                   |
| `text`           | I 形文本选择光标              | 𝄒                   |
| `move`           | 十字箭头（表示可移动）         | ✥                   |
| `wait`           | 等待/加载状态（通常是沙漏）    | ⌛                   |
| `help`           | 帮助问号                      | ❓                   |
| `not-allowed`    | 禁止操作（带斜线的圆圈）       | 🚫                   |
| `crosshair`      | 精确选择十字线                | ✚                   |
| `progress`       | 进度状态（通常是箭头+沙漏）    | ⏳                   |
| `grab`           | 抓取手势（张开的手）           | 🖐                   |
| `grabbing`       | 抓取中手势（握紧的手）         | 🤚                   |
| `col-resize`     | 列调整（左右箭头）             | ⇼                   |
| `row-resize`     | 行调整（上下箭头）             | ⇳                   |
| `zoom-in`        | 放大                          | 🔍+                  |
| `zoom-out`       | 缩小                          | 🔍-                  |
| `context-menu`   | 上下文菜单（箭头+菜单图标）     | ▤                   |
| `cell`           | 单元格选择（表格）             | ☐                   |
| `copy`           | 复制指示                      | ⎘                   |
| `alias`          | 别名/快捷方式创建              | ⤷                   |
| `no-drop`        | 禁止放置                      | 🚫                   |
| `vertical-text`  | 垂直文本选择（水平I型）        | ⇳                   |
| `all-scroll`     | 全方位滚动（四向箭头）         | ✥                   |
| `nesw-resize`    | 东北-西南方向调整              | ⇖                   |
| `nwse-resize`    | 西北-东南方向调整              | ⇗                   |
| `ew-resize`      | 东西方向调整                  | ↔                   |
| `ns-resize`      | 南北方向调整                  | ↕                   |
| `none`           | 无光标显示                    | (空白)               |
| `url()`          | 自定义光标图像                | (需指定图像路径)      |
### 特殊说明
1. **自定义光标**：可以使用 `url('path.cur'), auto` 格式引用自定义光标文件（支持 .cur 和 .png 格式）
2. **回退机制**：建议始终在自定义光标后添加通用光标值作为回退，例如：`url(cursor.png), pointer`
3. **浏览器支持**：部分光标样式（如 `zoom-in`、`grab`）需要现代浏览器支持
