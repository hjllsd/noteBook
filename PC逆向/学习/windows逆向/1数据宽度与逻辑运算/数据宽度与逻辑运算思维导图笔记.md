# 数据宽度
## 常见的概念
- 一个0或者1代表bit即位
## 常见数据宽度
- [[常见数据宽度]]
## 数据宽度的环绕性质
- 数据的宽度正最大位加一是负数的最小位 负数的最大位加一是正数的最大位
- 计算机只负责存储数据,读取有无符号是看读取者的想法
- 不是最高位是1才是负数,是因为对半分开刚还负数的最高为都是1
## 数据的宽度
- 位数的存储一旦超过当前的定义会抛弃高位的数据
# 逻辑运算
## 逻辑运算的分类

### 1. 与运算（AND）
- **符号**：`&`
- **描述**：只有两个操作数都为 `1` 时，结果才为 `1`，否则为 `0`。

### 2. 或运算（OR）
- **符号**：`|`
- **描述**：只要两个操作数中有一个为 `1`，结果就为 `1`，只有当两个操作数都为 `0` 时，结果为 `0`。

### 3. 非运算（NOT）
- **符号**：`!` 或 `~`（取决于具体语法）
- **描述**：对操作数进行取反，即将 `1` 变成 `0`，`0` 变成 `1`。

### 4. 异或运算（XOR）
- **符号**：`^`
- **描述**：当两个操作数不相同的时候，结果为 `1`，当两个操作数相同时，结果为 `0`。

### 5. 与非运算（NAND）
- **符号**：`NAND`
- **描述**：与运算的结果取反。

### 6. 或非运算（NOR）
- **符号**：`NOR`
- **描述**：或运算的结果取反。
