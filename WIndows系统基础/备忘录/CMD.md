## 介绍
CMD是WIndows的命令行工具,主要为一些命令行工具(无GUI)的工具提供运行环环境
## 常用命令

| 操作            | 说明                       |
| ------------- | ------------------------ |
| 盘符名称:         | 盘符切换。E:回车，表示切换到E盘。       |
| dir           | 查看当前路径下的内容。              |
| cd 目录         | 进入单级目录。cd itheima        |
| cd ..         | 回退到上一级目录。                |
| cd 目录1\目录2... | 进入多级目录。cd itheima\JavaSE |
| cd \          | 回退到盘符目录                  |
| cls           | 清屏。                      |
| exit          | 退出命令提示符窗口。               |
